# AI-Powered-Pet-Diagnosis-Tool
**AI-powered pet diagnosis tool (Certificate in ML at York University, group project)**

---

## Overview
This project is an AI-powered tool for predicting pet health symptom urgency using both structured (Excel) and unstructured (PDF/Word) medical records.  
It includes a trained AI model, interactive user interface, and API support.

---

## Deliverables
1. **Requirement Analysis & Specification Document**
2. **System Architecture & Design Documentation**
3. **AI Model**
   - Trained and validated model for symptom urgency prediction
   - Handles both structured and unstructured data
   - Tracks API usage monthly
   - Monitors model response & accuracy
4. **User Interface**
   - Interactive platform for symptom input and urgency prediction
5. **Deployment Plan & Documentation**
6. **User Manual & Training Materials**

---

## Features
- Deep learning-based symptom urgency prediction
- API using FastAPI
- Modular code structure
- Ready for Docker deployment
- Designed for both structured and unstructured pet medical data
